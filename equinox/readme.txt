This folder provides OSGI equinox development realated sample and project on Eclipse

---01_helloworld: offer helloworld sample to test environment setup.
---02_service: By using domo bundle, sample shows how to export service form provider, and import to consumer for test service.
---03_event: By using domo bundle, sample shows how to export event form provider, and import to consumer for test event communication.
---04_httpservice: demo to show how to embed httpservice to OSGI frame.
	--Declarative: construct OSGI http service through Declarative Registration
	--Programmatic: construct OSGI http service through Programmatic Registration
	--WebBoard: provide web application to show how to setup a equinox web development environment
